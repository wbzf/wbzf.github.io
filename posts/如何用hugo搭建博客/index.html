如何用Hugo搭建博客

  • Mac安装方式

    1. brew install hugo
    2. hugo version

  • Windows安装方式

    1. 去Hugo releases页面下载hugo_xxx_Windows-64.zip
    2. 解压 把hugo.exe放到D:\Software\hugo\hugo.exe
    3. 把D:\Software\hugo\加到PATH
    4. 重启终端 运行hugo version查看版本

  • 看文档搞出网站

    1. 进入Hugo官网 点击Quick Start快速开始
    2. 从Step2开始抄代码 直到Step7
    3. 得到一个public目录 这就是我们的博客站点
    4. hugo server -D可以预览草稿
    5. hugo server可以预览非草稿

  • 预览网站

    1. 双击打开public/index.html发现不能预览
    2. 因为public/index.html不能使用文件协议预览
    3. 请使用hugo server预览

  • 创建一个GitHub repo

    1. 仓库名为 你的github用户名.github.io
    2. 比如我的仓库名就是 wbzf.github.io
    3. 怎么查看自己的github用户名? 点头像 profile
    4. 创建的时候什么都不要勾选

  • 又一次创建本地仓库

    1. 进入public
    2. git init
    3. git add .
    4. git commit -m

  • 关联远程仓库

    1. git remote add origin git@github.com:xxxx/x.github.io.git
    2. 注意必须是git@开头
    3. git push -u origin master

  • 最后打开 x.github.io 的GitHub Pages
